## 12. Responsive Breakpoints

### sm (640px) - Large Mobile
- Single column maintained
- Text slightly larger
- Buttons slightly wider

### md (768px) - Tablet
- Two column layout possible
- Increased spacing

### lg (1024px) - Desktop
- Three column layout
- Full feature set visible
- All panels accessible
